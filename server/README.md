## 响应示例

```
//正确
{
    "errno": 0,
    "errmsg": "",
    "data": "bcc2866643bc037187047d4c6faac4de"
}
//错误
{
    "errno":1,
    "errmsg":"用户名已存在"
}
每个响应都为json字符串
errno为0时，都为请求成功
```

##  1.登录(admin)

### METHOD:POST

```
[path]/admin/login
```

### 请求报文

| 编号 | 字段     | 名称   | 类型   | 必须 | 描述 |
| ---- | -------- | ------ | ------ | ---- | ---- |
| 1    | name     | 用户名 | String | M    |      |
| 2    | password | 密码   | String | M    |      |

### 响应报文

| 编号 | 字段    | 名称    | 类型   | 必须 | 描述 |
| :--- | ------- | ------- | ------ | ---- | ---- |
| 1    | session | session | String | M    |      |
